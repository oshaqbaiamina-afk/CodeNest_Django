Проектті жүргізбей тұрып, алдымен codenest_project папканың ішіндегі settings ті өздеріңіздің Деректер қорларыңызға сілтемені қалдырыңыз, құпия сөз, порттың дұрыстығына қол жеткізіңіз. Сосын терминалда алдымен виртуалды әлемді(venv) кетіріп, 
жаңадан жасап қажетті Lib терді жүктеңіз, Django ның болуына көз жеткізіңіз. Тек содан кейін осы командаларды терминалда орындаңыз...
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass 
бұл уақытша шектеулерді алып тастауға қажет, файлдарыңыздың қауіпсіздігін сактау үшін 
.\venv\Scripts\Activate.ps1
виртуалды жерді іске қосу үшін
python manage.py runserver
бағдарламаны іске қосу үшін 

python manage.py assign_students
оқушыларды мұғалімдерге автоматты түрде тіркеу үшін

python manage.py backup_database --format=json --compress
backup жасау үшін
